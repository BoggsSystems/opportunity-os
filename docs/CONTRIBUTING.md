# Contributing

## How to Contribute

We welcome contributions to Opportunity OS! Here's how to get started:

### Prerequisites

- Node.js 18+
- pnpm
- Docker
- Basic knowledge of TypeScript, Node.js, and databases

### Setup

1. Fork the repository
2. Clone your fork
3. Run `./scripts/setup.sh`
4. Create a feature branch

### Development Workflow

1. Make your changes
2. Test thoroughly
3. Ensure code quality
4. Submit a pull request

### Code Quality

- Run `pnpm lint` to check linting
- Run `pnpm typecheck` to verify types
- Run `pnpm test` to run tests
- Follow existing code patterns

### Submitting Changes

1. Create a descriptive branch name
2. Write clear commit messages
3. Include tests for new features
4. Update documentation as needed
5. Submit a pull request with description

### Areas for Contribution

- **Frontend**: React/Next.js web application
- **AI Integration**: New AI providers and prompts
- **Integrations**: External service connections
- **Browser Automation**: Playwright scripts
- **Documentation**: Guides and examples
- **Testing**: Unit and integration tests

### Guidelines

- Keep changes focused and minimal
- Follow existing patterns and conventions
- Write meaningful commit messages
- Include tests for new functionality
- Update relevant documentation

### Getting Help

- Check existing issues and discussions
- Create an issue for bugs or questions
- Join discussions for feature ideas
- Review existing pull requests for patterns

Thank you for contributing!
