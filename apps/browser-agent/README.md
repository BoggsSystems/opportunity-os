# Opportunity OS Browser Agent

Browser automation agent for Opportunity OS using Playwright.

## Development

This app handles web scraping, form filling, and browser automation tasks.

## Structure

- `src/main.ts` - Entry point
- `src/` - Source code

## Scripts

- `pnpm build` - Build the application
- `pnpm start` - Start the agent
- `pnpm start:dev` - Start in development mode
